# SpellcastingImprovements

**Version:** 1.0.2  
**Author:** the-xorcist  
**Release Date:** February 15, 2026

## Description

A quality-of-life mod for Erenshor that enhances the spellcasting system with fixes for spell stacking, buff management, and movement speed mechanics.

## Features

- **Spell Stacking Fix**: Automatically removes lower-level spells when casting higher-level replacements on the same spell line
- **Click-Off Buffs**: Click on buff icons to remove beneficial spells from your character (works on any beneficial buff, including those cast by SimPlayers)
- **Spell Message Fix**: Prevents misleading "spell took hold" messages when a spell is blocked by a higher-level effect
- **Backward Movement Speed**: Allows movement speed buffs to increase backward movement speed cap (configurable percentage)

## Installation

1. Install [BepInEx](https://docs.bepinex.dev/articles/user_guide/installation/index.html) for Erenshor
2. Copy `SpellcastingImprovements.dll` to your `BepInEx/plugins` folder
3. Launch the game
4. (Optional) Copy `SpellcastingImprovements.cfg` to your `BepInEx/config` folder for preset defaults
